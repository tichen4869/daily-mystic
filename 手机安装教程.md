# 每日玄学 · 手机安装教程

## iPhone 用户

### 一、添加到主屏幕（像 App 一样用）

1. 用 **Safari** 打开：https://daily-mystic.onrender.com
2. 点底部中间的 **分享按钮**（一个方框带向上箭头 ↑）
3. 往下滑，找到 **Add to Home Screen**（添加到主屏幕）
4. 点 **Add**（添加）
5. 主屏幕就多了一个「每日玄学」图标，以后点它就能看运势

### 二、设置每天自动语音播报（可选）

1. 打开 **Shortcuts**（快捷指令）App
2. 点右上角 **+**
3. 点 **Add Action** → 搜索 **Get Contents of URL** → 选它
4. 点 URL 那一栏，粘贴：
   ```
   https://daily-mystic.onrender.com/api/voice?birth=你的生辰
   ```
   把「你的生辰」替换成你的出生时间，格式：年/月/日/时:分
   例如：`https://daily-mystic.onrender.com/api/voice?birth=1994/04/21/03:14`

5. 点底部 **搜索栏** 添加下一步 → 搜索 **Get Dictionary Value** → 选它
6. 点 Key，输入：`text`
7. 再添加 → 搜索 **Speak Text** → 选它（会自动连上上一步）
8. 再添加 → 搜索 **Show Notification** → 选它
   - Title 填：`每日玄学`
   - Body 选：Dictionary Value
9. 点顶部的名字，改为 **每日玄学**
10. 点 **Done**

#### 设置每天自动运行：
1. 点底部 **Automation**
2. 点 **+** → 选 **Time of Day**
3. 选早上 **8:00 AM** → **Daily**
4. 点 **Next**
5. 选刚才的 **每日玄学**
6. 关掉 **Ask Before Running**
7. 点 **Done**

搞定！每天早上 8 点自动语音播报 + 推送通知 ✧

## Android 用户

### 一、添加到主屏幕

1. 用 **Chrome** 打开：https://daily-mystic.onrender.com
2. 点右上角 **三个点 ⋮**
3. 选 **添加到主屏幕** / **Add to Home screen**
4. 点 **添加**
5. 主屏幕就有图标了

### 二、每天提醒（可选）

手机自带的 **时钟** App → **闹钟** → 设一个早上 8 点的闹钟
标签写「看运势」，每天响了点一下主屏幕图标就行

---

> 首次打开点「命盘」输入你的生辰就能用了
> 可以添加多个人的生辰，点 + 号切换
